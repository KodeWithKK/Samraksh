# Project Setup Guide

## Backend

- Install Docker

- Build Image (Don't run for subsequent use)

  ```shell
  docker-compose build
  ```

- For Running

  ```shell
  docker-compose up -d
  ```

- For Stopping

  ```shell
  docker-compose down
  ```

## Frontend

- Install nodejs

- open frontend directory on terminal and write the following commands

  ```bash
  npm install
  ```

  ```bash
  npm run dev
  ```

## Output

- See it on http://localhost:5173/
