# Project Setup Guide

## Backend

- Install python 3.12.5

- Open backend directory on terminal and run following commands

  ```bash
  python -m venv .venv
  ```

  ```bash
  pip install poetry
  ```

  ```bash
  poetry shell
  ```

  ```bash
  poetry install
  ```

  ```bash
  python main.py
  ```

## Frontend

- Install nodejs

- open frontend directory on terminal and write the following commands

  ```bash
  npm install
  ```

  ```bash
  npm run dev
  ```

## Output

- See it on http://localhost:5173/
